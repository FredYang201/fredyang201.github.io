Snake game

'Snake' is a game written in JavaScript. Users can start the game by click 'start game' button and control the movement of snake by pressing the arrow keys on the keyboard. Goal of the game is to eat apples as many as possible. Once the head of snake hits the wall or the body of snake itself, the game will be over.
